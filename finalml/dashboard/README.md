# Advanced ML Ablation Dashboard

A standalone dashboard for visualizing ML ablation study results from both normal and closed datasets.

## Features

- **Mode Selection**: Switch between Normal and Closed dataset results
- **Model Selection**: Choose between SVM, GBC, and XGB models
- **Feature Ablation**: Select/deselect environmental features (Temperature, Humidity, VOC)
- **Real-time Results**: View accuracy, precision, recall, and F1 scores
- **Responsive Design**: Works on desktop and mobile devices

## Setup

1. Install dependencies:
```bash
cd dashboard
npm install
```

2. Start the server:
```bash
npm start
```

3. Open browser and navigate to:
```
http://localhost:3001
```

## API Endpoints

- `GET /api/ml/ablation-svm` - Returns normal dataset ablation results
- `GET /api/ml/ablation-closed` - Returns closed dataset ablation results

## File Structure

```
dashboard/
├── server.js          # Express server with API endpoints
├── index.html         # React dashboard (single file)
├── package.json       # Node.js dependencies
└── README.md         # This file
```

## Usage

1. Select dataset mode (Normal/Closed)
2. Choose models to compare
3. Select environmental features for ablation study
4. Click "Calculate" to see results
5. View performance metrics in card format

## Data Sources

The dashboard reads ablation results from:
- `../svm (2)/svm/unified_outputs/ablation_results.csv` (Normal)
- `../svm (2)/svm/closed/ablation_results.csv` (Closed)

Make sure these CSV files exist and are generated by running the respective pipelines.
